OBS: ENUMS finns inte i JavaScript. Man kan uppnå samma funkionalitet genom att göra ett Object literal och sätta 
Object.freeze vilket ger ett låst objekt som  inte går att ändra på eller extenda.

Basically const Colors = Object.freeze({
  RED: 'red',
  BLUE: 'blue',
  GREEN: 'green'
});

Dock så finns typen "enum" i TypeScript.
